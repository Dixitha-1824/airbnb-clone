# airbnb-fullstack-clone
A full-stack Airbnb-style web application built to understand real-world backend and authentication workflows.

##  Live Demo
  https://airbnb-clone-3mzk.onrender.com

## Features

- User Authentication (Signup / Login / Logout)
- Secure Password Hashing using Passport.js
- Create, Edit & Delete Listings
- Image Upload using Cloudinary
- Reviews & Ratings System
- Authorization (Only owners can edit/delete)
- Flash messages for user feedback
- MongoDB Atlas integration
- Deployed on Render

## Tech Stack

- EJS
- Bootstrap 5
- EJS
- Node.js
- Express.js
- MongoDB & Mongoose
- Passport.js (Authentication)
- Cloudinary (Image storage)

This project was built for learning purposes and follows industry-level backend practices.
